Summary: Latitude VS. X

During December 11,2017, our Data Bootcamp was asked to get information off of a Google Weather API. The graphs and information demonstrated show a full stack use of analysis. In this website are demonstrated some of the key elements of data analysis including : Gathering data from a source (Google API), using a dataframe in order to organize our information( The Python Library pandas), and a graphing system in order ot display our information (The Python Library matplotlib).I hope you enjoy this demonstration.
